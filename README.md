## 🧰 Framework da Tecnologia:

### Teoria dos Grafos:

- A teoria dos grafos é um ramo da matemática e ciência da computação que estuda relações entre objetos, modelados como grafos, formados por vértices (nós) e arestas (conexões). Utilizada para resolver problemas de conectividade, caminhos mínimos e redes (sociais, logísticas, entre outros), esta teoria, iniciada por Euler, abstrai situações complexas em estruturas visuais.

````
Método para organizar os dados em forma e grafos.
````

### Entendendo base de dados:

````
Compreender e analisar os dados de forma prática, entre outros.
````

### Transformando dados e informações:
````
Diferença entre dados e informações, dados valor qualitativo ou quantitativo diferente da informação que responde uma pergunta de negócio (dúvida) com cálculos para análise específica.
````
### A importância da modelagem:

````
Essencialmente, a modelagem funciona como um ensaio estratégico: ela traduz ideias em estruturas visíveis para testar hipóteses, corrigir falhas e refinar o desempenho antes de investir recursos na execução real.
````
### Grafos e suas propriedades:

````
 🔵 Nodes, {nós: [(entidades)/(objetos)]}
 🔗 Relationships, {relações: [(conexões/ações)]}
 🏷️ Labels; & {rótulos: [(categorias)/(classes)]}
 📑 Properties. {propriedades: [(atributos/detalhes)]}
````
### Modelagem de dados em grafos:

````
A modelagem de dados em grafos é o processo de organizar informações priorizando as conexões entre os dados tanto quanto os próprios dados. Diferente do modelo relacional tradicional (SQL - Structured Query Language), que utiliza tabelas e colunas, a modelagem em grafos reflete como pensamos naturalmente sobre relacionamentos no mundo real.
````

### Trabalhando com Big Numbers:

`````
São informações apresentados sob forma numérica, com grande destaque no contexto de dashboards. Eles respondem perguntas de negócios de forma simples para orientar a tomada de decisões baseadas em dados. 
`````
